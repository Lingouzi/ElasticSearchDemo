# ElasticSearchDemo
springmvc + spring-data-elasticsearch 的 demo
> elasticsearch版本号: 7.6.2, 这个版本号是强制要求, 
>不然会出现各种奇奇怪怪的问题  

如需使用其他elasticsearch版本, 请对应修改 `spring` 版本和 `spring-data-elasticsearch` 版本
现有分支: 

| branchName   | esVersion |
| ------------ | --------- |
| master       | 6.8.6     |
| elastic7.6.2 | 7.6.2     |
